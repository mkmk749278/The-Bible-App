# Play Console — Content rating (IARC) questionnaire answers

Use in **Play Console → App content → Content ratings**. Expected result: **Everyone /
PEGI 3 / Rated for 3+** (a reference app with religious text).

- **Category:** Reference, News, or Educational → choose **Reference / Book**.
- **Violence** (cartoon, realistic, blood, etc.): **No** to all.
  - Note: the Bible text contains historical accounts; IARC treats scripture/reference
    text as non-interactive reference content. Answer the *interactive game/app
    behaviour* questions, all **No**.
- **Sexuality / nudity:** **No.**
- **Profanity / crude humour:** **No.**
- **Controlled substances (drugs, alcohol, tobacco):** **No.**
- **Gambling (simulated or real):** **No.**
- **Fear / horror:** **No.**
- **User-generated content / social features / sharing:** the app can share a verse
  image to other apps, and stores personal notes locally. There is **no in-app social
  network, chat, or user-to-user content**. Answer "Does the app let users interact or
  exchange content?" → **No** (sharing to the OS share sheet is not in-app interaction).
- **Personal information shared:** **No.**
- **Digital purchases:** **No.**
- **Location sharing:** **No.**

Religion/ideology is collected by the questionnaire for context only and does not raise
the age rating.
